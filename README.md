CB-Logbuch v1.0.1 - Programminformationen
Stand: 18.04.2026
=========================================

Dieses Programm ist ein spezialisiertes Logbuch-Tool für CB- und Amateurfunk-Enthusiasten. 
Es ermöglicht die effiziente Erfassung, Verwaltung und den Export von Funkkontakten (QSOs).

Hauptfunktionen:
----------------
* Datenbank-Management: Lokale Speicherung aller QSOs in einer SQLite-Datenbank.
* CAT-Steuerung: Unterstützung von Hamlib (rigctld) zur automatischen Frequenz- und Modusübernahme vom Funkgerät.
* Auto-Import: Überwachung von ADIF-Dateien (z.B. von WSJT-X oder JS8Call) für automatisches Live-Logging.
* Locator-Tools: Automatische Berechnung der Entfernung zwischen zwei QTH-Locatoren.
* Export-Optionen: Unterstützung für das ADIF-Format (Standard für Funk-Logs) und Excel-kompatible CSV-Exporte.
* Backup-System: Automatisierte Sicherungen der Datenbank beim Beenden des Programms.
